# Measure Latency Using TCP

In Python, we can use `tcp-latency` module to measure Latency using TCP.

## Example

```python
>>> from tcp_latency import measure_latency
>>> measure_latency(host='google.com')
[103.44290733337402]
```

## Installation

```bash
pip install tcp-latency
```
