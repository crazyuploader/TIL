# Custom Shell Prompt

`StarShip` is Cross-Shell Prompt!

**Quick Installation**

```bash
$ curl -fsSL https://starship.rs/install.sh | bash
```

**Adding to Bash**

Add the following to the end of `~/.bashrc`:

```bash
eval "$(starship init bash)"
```
