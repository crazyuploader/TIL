# Install Pip

> pip - A tool for installing and managing Python packages

To install Python package installer `pip` in Arch Linux, simply execute the following command -

## Pip for Python 3

```bash
$ sudo pacman -S python-pip
```

## Pip for Python 2

```bash
$ sudo pacman -S python2-pip
```
